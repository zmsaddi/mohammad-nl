# نظام المحاسبة - Accounting System

نظام محاسبة عربي متكامل لإدارة المبيعات والمشتريات والمصاريف والتوصيل.

## المميزات

- شراء وبيع البضائع مع تتبع المخزون
- نظام توصيل تلقائي عند كل بيع
- نظام ديون للعملاء مع تسجيل الدفعات
- داشبورد شامل مع رسوم بيانية
- فصل المدفوعات (نقدي / بنك)
- قاعدة بيانات ذكية (منتجات، موردين، عملاء)
- تحميل البيانات كملف Excel
- تسجيل دخول مع صلاحيات (مدير / موظف)
- تصميم عربي RTL متوافق مع كل الشاشات

## التقنيات

- Next.js 16 (App Router)
- Vercel Postgres (Neon)
- NextAuth.js
- Tailwind CSS
- Recharts
- SheetJS (xlsx)

## التشغيل

```bash
npm install
npm run dev
```

بيانات الدخول الافتراضية: `admin` / `admin123`
